A **Contributor** is another Scribe author who can write articles for your site. Contributors can create and edit articles under your site, but you remain the Site Owner — only you can manage site settings, create and rename groups, and invite or remove contributors.

## What contributors can do

| Action | Owner | Contributor |
| ------ | ----- | ----------- |
| Write and edit articles | ✓ | ✓ |
| Publish articles to a group | ✓ | ✓ |
| Create and rename groups | ✓ | — |
| Change site settings | ✓ | — |
| Invite contributors | ✓ | — |
| Remove contributors | ✓ | — |

## Coming soon

The contributor invitation flow is planned for a future release. When it ships, you'll be able to invite anyone with a Bluesky account to contribute to your site directly from Scribe CMS.
