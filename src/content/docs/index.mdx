## What is Scribe ATP?

Scribe ATP is an open-source publishing stack built on the [AT Protocol](https://atproto.com) — the same open protocol that powers Bluesky. Your content lives in your own account and can power any number of websites simultaneously.

<CardGrid>
  <Card title="Scribe CMS" icon="pencil">
    The writing tool. Compose articles, organise them into sites and groups, and control what's published — at [scribe-cms.app](https://scribe-cms.app).
  </Card>
  <Card title="Scribe SDK" icon="puzzle">
    The developer toolkit. TypeScript packages for React, Angular, Vue, Next.js, Nuxt, React Router, and plain TypeScript. Fetch your content in minutes.
  </Card>
  <Card title="For authors" icon="open-book">
    Write and publish without handing your content to a third-party platform. You own everything — no lock-in.
  </Card>
  <Card title="For developers" icon="laptop">
    One SDK, any framework. Add content to an existing site or build a new one from scratch. Full TypeScript support throughout.
  </Card>
</CardGrid>
